# 📊 SaaS Dashboard (Work in Progress)

A simple SaaS-style admin dashboard built as part of my full-stack learning journey.  
This project focuses on building a real backend system with authentication, authorization, and basic data management.

> ⚠️ This project is still under active development.

---

## 🚀 Tech Stack

- Node.js
- Express
- MongoDB
- JWT Authentication

---

## ✨ Current Features

- 🔐 **Authentication**
  - Login with JWT
  - Protected routes

- 🛡️ **Role-Based Access Control (RBAC)**
  - Different access levels (e.g. admin / user)

- 📦 **Product CRUD**
  - Create, read, update, delete products

- 👥 **User CRUD**
  - Basic user management

- 📊 **Dashboard Endpoint**
  - Returns basic stats:
    - Total users
    - Total products
